# 📄 Day 3 – HTML Resume Project

## Overview
This project is part of my MERN Stack learning journey.  
On Day 3, I built a personal resume using **pure HTML**, focusing on **semantic structure, clean organization, and readability**.  

The goal was to create a well-structured resume webpage that demonstrates foundational HTML skills.

## What I Learned
- How to structure a webpage using HTML
- Using semantic and basic HTML tags:
  - `<section>` for content grouping
  - `<table>` for layouts
  - `<ul>` and `<li>` for lists
  - `<a>` for links
  - `<img>` for images
  - `<details>` and `<summary>` for interactive FAQs
- Organizing content into sections:
  - Contact Information
  - Bio
  - Education
  - Work Experience
  - Skills
  - Achievements
  - Projects
  - FAQ
- Best practices for clean and readable HTML

## Features
- Displayed personal contact information with clickable links (email, LinkedIn, GitHub)
- Professional bio section
- Education and work experience lists
- Skills and achievements lists
- Projects with hosted and GitHub links
- Interactive FAQ section using `<details>` and `<summary>`
- Footer with copyright notice
