# 🎨 Day 5 – CSS Basics

## 🚀 Overview

This project is part of my MERN Stack learning journey.
On Day 5, I focused on understanding the **basics of CSS styling** and how to apply styles to HTML elements using an external stylesheet.

## 🧠 What I Learned

* How to link an external CSS file using `<link>`
* Difference between **id selectors** and **class selectors**
* Styling text using:

  * `font-family`
  * `font-size`
  * `font-weight`
* Applying:

  * Background colors
  * Borders
  * Width, padding, and margin

## 💻 Features

* Styled multiple paragraphs with different background colors
* Applied consistent layout using reusable classes
* Practiced spacing using margin and padding
* Created a simple and clean UI structure
