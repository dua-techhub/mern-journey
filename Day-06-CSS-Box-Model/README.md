# 📦 Day 6 – CSS Box Model

## 🚀 Overview

This project is part of my MERN Stack learning journey.
On Day 6, I focused on understanding the **CSS Box Model** and how spacing works around elements.

The goal was to see how `margin`, `padding`, `border`, and `content` interact to control layout.

## 🧠 What I Learned

* The structure of the **CSS Box Model**:

  * Content
  * Padding
  * Border
  * Margin

* Difference between:

  * Block elements (`div`)
  * Inline elements (`span`)
  * Inline-block elements

* How spacing affects layout and alignment

* How `box-sizing: border-box` helps control element size

## 💻 What I Built

* Multiple `div` elements styled with borders and spacing
* A special box to clearly show the effect of large padding
* A `span` element converted into a box using `display: inline-block`
* Basic layout adjustments using margin and padding
