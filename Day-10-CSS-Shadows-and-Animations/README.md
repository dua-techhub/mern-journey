# 🎬 Day 10 - CSS Shadows & Animations

This project is part of my MERN stack learning journey.
In this practice, I explored **CSS animations and basic visual effects** by creating moving elements and a bouncing shape.

## 📁 Project Overview

The project includes:

- 🟦 Two animated boxes
- 🟣 One animated circle
- 🎨 Styled container layout
- 🔁 Smooth animations using CSS `@keyframes`

---

## ✨ Features

- **Box 1** → Moves from left to right
- **Box 2** → Moves from right to left
- **Circle** → Bounces vertically
- Infinite animation loops
- Smooth motion using `ease-in-out`

---

## 🧠 What I Learned

- How to use `@keyframes` for animations
- Using the `animation` shorthand property
- Controlling movement with:
  - `left`, `right`
  - `transform: translateY()`

- Creating shapes like circles using `border-radius`
- Structuring animated layouts with `position`

---

## 🚀 How to Run

1. Download or clone the repository
2. Open the HTML file in your browser
3. Make sure all code is in the same file or properly linked
