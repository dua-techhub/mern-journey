# 📄 Day 4 – Media, Favicon & iFrames

## Overview
This project is part of my MERN Stack learning journey.  
On Day 4, I focused on enhancing webpages with **media elements**, including:

- Adding a favicon  
- Using meta tags and title  
- Embedding audio and video  
- Using iFrames to embed external content (YouTube videos)  

The goal was to make a webpage more interactive and professional.

## What I Learned
- How to add a favicon using `<link rel="shortcut icon">`  
- The importance of `<meta>` tags for responsiveness and SEO  
- Embedding audio with `<audio>` and controls  
- Embedding external content with `<iframe>`  
- Structuring multiple videos using tables for layout  

## Features
- Custom favicon for the webpage  
- Responsive viewport meta tag  
- Audio player embedded in the page  
- Multiple sections with embedded YouTube videos:  
  - Roadmap Videos  
  - Operating System Videos  
  - Database Videos  
- Navigation links to quickly jump to different sections

## Key Concepts
- Media elements like audio and video improve engagement on webpages  
- Favicon and meta tags improve branding and responsiveness  
- iFrames allow embedding external content seamlessly  
- Organizing media in sections with tables helps maintain layout  

## Output
- Webpage displays header, bio, and navigation menu  
- Audio player for song playback  
- Sections with embedded YouTube videos for Roadmap, OS, and DBMS topics  
- Favicon visible in the browser tab
