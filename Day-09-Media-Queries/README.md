# 📦 CSS Layout Practice (Media Queries & Boxes)

This project is part of my learning journey in CSS. It focuses on practicing **basic layout structures** and preparing for **responsive design using media queries**.

## 📁 Project Structure

- Multiple HTML files demonstrating:
  - Container with multiple boxes
  - Simple single box layout

- External CSS file (`style.css`) used for styling

## 🧱 Features

- Created a container with multiple boxes:
  - Box 1
  - Box 2
  - Box 3
  - Box 4

- Practiced reusable `.box` class
- Linked external stylesheet
- Set up viewport for responsive design

## 💡 What I Learned

- How to structure HTML layout using `<div>`
- Using **class** and **id** selectors
- Linking CSS file to HTML
- Importance of:

  ```html
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  ```

- Basics before applying **media queries**

## 🚀 How to Run

1. Download or clone the repository
2. Open any HTML file in your browser
3. Make sure `style.css` is in the same folder
